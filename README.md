# LaMenu
A menu of connected shortcuts to be used for opening series of links at the same time, and track how many times you have opened that prefab of applications

In the file BackupScores.txt You can edit the presets you want, for example I have
Game,0,Work,0,Music,0 

This is incase something happens to the Score.txt file which will be generated on first use

Inorder to add a set of links to open you can add text files under the name of the preset with the Directory of your chosen links to open.
For instance:

Under Game.txt in the Links Folder there is E:(directory)steam.exe,E:(directory)spotify.exe
seperated by commas
